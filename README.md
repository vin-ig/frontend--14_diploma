# Дипломный проект

В данном проекте реализован сайт веб-студии.

На главной странице есть некая справочная информация о студии, предлагаемые услуги, контакты и др.
Пользователь может кликнуть на заинтересовавшую услуги или акцию, после чего ему будет предложено оставить свои контакты
для последующей связи. На сайте есть блог со статьями, на который пользователь может перейти из шапки, футера или
по ссылке на главной странице.

В приложении реализован функционал аутентификации и авторизации пользователя. Авторизованные пользователи могу оставлять
комментарии к статьям, а также оценивать их или отправлять жалобу.

Также сайт поддерживает адаптивную версию для мобильных устройств.

___

## Запуск проекта

1. Клонирование репозитория `git clone https://github.com/vin-ig/frontend--14_diploma.git`.
2. Запуск MongoDB, например `d:/mongodb/bin/mongod` или другой командой в зависимости от расположения клиента.
3. Переход в папку backend `cd backend` (в новом терминале)
4. Установка зависимостей (при первом запуске проекта) `npm install`
5. Запуск миграций (при первом запуске проекта) `migrate-mongo up`
6. Запуск сервера backend `npm start`
7. Переход в папку frontend `cd frontend` (в новом терминале)
8. Установка зависимостей (при первом запуске проекта) `npm install`
9. Запуск сервера frontend `ng serve --open`
